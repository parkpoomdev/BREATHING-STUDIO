## ‚ú® Features

### üé® Visual Customization
- **4 Meditation Palettes**: Ocean (blue), Forest (green), Sunset (pink), Zen (purple)
- **Light/Dark Themes**: Switch between light and dark modes for any environment
- **Format Ratios**: Choose between 16:9 (horizontal) or 9:16 (vertical) aspect ratios

### üå¨Ô∏è Breathing Patterns
- **Relaxation (4-4-8)**: 4s inhale ‚Üí 4s hold ‚Üí 8s exhale
- **Box Breathing (4-4-4-4)**: 4s inhale ‚Üí 4s hold ‚Üí 4s exhale ‚Üí 4s hold
- **Deep Calm (4-7-8)**: 4s inhale ‚Üí 7s hold ‚Üí 8s exhale

### üìä Visual Feedback
- Animated breathing circle that scales with inhale/exhale
- Clock ring with progress dots around the perimeter
- **Phase markers**: Emphasized larger circles at breathing phase transitions
- Real-time phase text (Inhale/Hold/Exhale)
- Smooth transitions and animations

### üíæ Export Functionality
- Export animated SVG files for use in presentations, videos, or other projects
- Maintains all animations and colors in the exported file
- Supports both horizontal and vertical formats

## üöÄ Getting Started


2. Open `index.html` in your web browser:

That's it! No build process or dependencies required.

## üéØ How to Use

1. **Choose Your Settings**:
   - Select a theme (Dark/Light)
   - Pick a format ratio (16:9 or 9:16)
   - Choose a meditation palette color
   - Select a breathing pattern

2. **Start Your Session**:
   - Click "START SESSION" to begin breathing
   - Follow the visual cues:
     - Circle expands = Inhale
     - Circle holds = Hold breath
     - Circle contracts = Exhale
   - Watch the clock ring dots to track progress

3. **Export Your Animation**:
   - Click "EXPORT" to download an animated SVG file
   - Use it in presentations, videos, or share with others
